Control flow and loops

Control flow is the order which the computer executes code, top to bottom.

A loop is a sequence of instructions that is repeated until a certain condiiton has been met.

In other words, you want to go for a walk, but you can't go for a walk without socks and shoes on. To continue your control flow, you must complete the loop of putting 1 sock on each foot, then 1 shoe on each foot and you're ready to continue your day!

What is the DOM?

The DOM or Document Object Model treats the webpage on your browser as a tree so you can see all of the branches (paragraphs (also divs), headings, titles, footers, colours etc) within one interface.

If you wanted to, you can change things like headings or text straight from the DOM itself! Without having to load up files and put them in a code editor, it can be done on the browser itself.

You can use the DOM on any web page, however it will only stay changed on your local page until you refresh the page. Then it will change back to its original heading/text.

Accessing data from arrays and objects

Arrays have information that display items via index under one name/variable. e.g

favCandy=["chocolate", "sour", "lollipops"]

Objects have information based on one key and one value. e.g

myCat= {

name="Zelda"

breed="Tabby"

year="2016"

}

So I could ask the computer for the second (technically it's considered 1st position because array's start at 0 but lets not get into it right now) favourite candy and it would tell me sour, but I cannot ask the computer for the second value of myCat as it is programmed by the key and not the index.

What's a function? Why is it helpful?

A function is a list of instructions you give the computer to do something, through JavaScript.

Functions are helpful because it gives web developers a small amount of code that can be re-used, whether it's complicated with extra loops and conditionals (lets not talk about it) or you want it to say hello! It also helps to keep code organised if you need to de-bug or fix some code you made hours earlier, instead of getting your reading glasses out and trawling through everything you can have a better idea where everything is.
