Problem Solving

A time I was blocked on a simple problem

I was working on JS Intro Operators where it was asking to make a function to plus some numbers together, so I did that! It wanted 3 + 7 so I wrote out

function addNum(num1, num2) {

return 3 + 7 }

But then it wanted 11 + 2? How am I supposed to get 3 + 7 and 11 + 2 give me the same answer? It's just not possible. I was stumped.

Until... after some thought, I took a step back and read the question again. Then, I realised, they have the math equations already written out somewhere that I can't see, I'm not supposed to make it literal. They just want a placeholder for the questions to come into effect. e.g

function addNum(num1, num2) {

return num1 + num2 }

Suddenly all of the questions turned green and I felt silly. Sometimes you truly need to take a step back and look at the bigger picture.

Instead of getting hung up on some numbers....

A time I solved a problem elegantly

Would you believe that once I took a step back from the Operators, and looked at it as a whole, it clicked?

Next question they asked to subtract numbers. BOOM, I knew that! It became

return num1 - num2

Same for division!

return num1 / num2

And multiplication

return num1 * num2

Basic maths, we got there team. Smooth as butter.
